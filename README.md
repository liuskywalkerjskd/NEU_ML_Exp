It's recommended to use Anaconda for managing the Python environment. Installation guides are provided in the course materials.

```bash

conda create -n ml_exp python=3.8

# Activate the environment
conda activate ml_exp

# Install required packages
pip install numpy pandas matplotlib scikit-learn jupyter
```
